ПРОЕКТ: ИГРА В ШАШКИ

1. ОСНОВНЫЕ КОМПОНЕНТЫ
---------------------------------
- CheckersGame: Главный класс игры
- board: 8x8 матрица игрового поля
- current_player: Активный игрок ('W'/'B')
- initialize_board(): Стартовая расстановка
- print_board(): Визуализация доски
- is_valid_position(): Проверка формата позиции
- convert_position(): Конвертация "e2" → (6,4)
- get_moves(): Расчет допустимых ходов
- has_captures(): Проверка обязательных взятий
- make_move(): Обновление состояния доски
- play(): Управление игровым процессом

2. ПРАВИЛА РЕАЛИЗАЦИИ
---------------------------------
- Доска: 8x8 клеток, шашки занимают черные поля
- Старт: Белые (верх), Черные (низ)
- Движение:
  • Обычные: по диагонали вперед
  • Дамки: по диагонали в любом направлении
- Взятие: Прыжок через вражескую шашку
- Обязательное взятие: При наличии возможности
- Превращение: В дамку на последней линии
- Победа: Уничтожение всех шашек противника

3. АРХИТЕКТУРНЫЕ РЕШЕНИЯ
---------------------------------
- Инкапсуляция: Вся логика в одном классе
- Двумерный массив: Простое представление состояния
- Модульность: Разделение на логические функции
- Координатная система: Буквенно-цифровая (a1-h8)
- Валидация: Проверка корректности всех ходов
- Чередование ходов: Автоматическая смена игроков

4. ИГРОВОЙ ПРОЦЕСС
---------------------------------
1. Белые ходят первыми
2. Ввод хода в формате: "начало конец" (e2 e3)
3. Автоматическая проверка:
   - Корректность позиций
   - Возможность хода
   - Обязательность взятия
4. Автоматическое:
   - Перемещение шашек
   - Удаление сбитых шашек
   - Превращение в дамки
   - Определение победителя

5. ЗАПУСК ИГРЫ
---------------------------------
> python checkers.py

6. КЛЮЧЕВЫЕ ОСОБЕННОСТИ
---------------------------------
- Полное соответствие правилам русских шашек
- Интуитивный консольный интерфейс
- Подробная валидация ввода
- Автоматический контроль правил
- Четкая визуализация игрового состояния
